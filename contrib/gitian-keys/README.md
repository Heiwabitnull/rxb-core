
## PGP keys of RXB contributors



   This file (keys.txt) can be used to list fingerprints of PGP keys
   for contributors to the RXB project.

   These keys are optional and are used to sign commits or build results.
   If you want to contribute to RXB and provide signed commits
   you may add your key to keys.txt following the format:

   FINGERPRINT Name Surname

   To fetch or refresh keys, you can use:

   gpg --refresh-keys

   Add your key if you plan to sign commits or builds for RXB.
